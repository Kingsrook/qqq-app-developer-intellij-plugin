This template creates a blank Liquibase changelog XML file.
Once you're inside the file, you will have a "Good Time" if you use some of the live templates that we provide, such as:
  • changeSetSQL - new change set with an SQL statement as its body.
  • changeSetLoadData - new change set for loading data from a CSV file as its body.
  • createTable - CREATE TABLE statement, with standard bits (id, create_date, modify_date) filled in.
  • alterAddColumn - ALTER TABLE statement to add a column.
  • alterAddIndex - ALTER TABLE statement to add an index.
  • alterAddUnique - ALTER TABLE statement to add a UNIQUE index.
