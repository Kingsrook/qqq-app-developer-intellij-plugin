This template creates a new QQQ RecordEntity class, with a TABLE_NAME constant, constructors, and stadard fields (id, createDate, modifyDate).
Once you're inside the file, you will have a "Good Time" if you add the rest of the fields in your table (the qfield live tempalte can help a little).

Then, to write your getters, setters, and fluent-setters ("withers"), you may like to select your fields, and run the GetSetWith tool from the Kingsrook Commentator plugin (under Tools > Kingsrook Commentator).
